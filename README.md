# Análisis de Calidad del Aire y Monitoreo Ambiental (CABA - ACUMAR)

## 📋 Descripción del Proyecto
Este proyecto desarrolla un análisis integral sobre la evolución de la calidad del aire en la Ciudad Autónoma de Buenos Aires (CABA), integrando diversas fuentes de datos abiertos para identificar patrones, tendencias y correlaciones ambientales. 

El enfoque principal combina el rigor estadístico con la visualización de datos para transformar registros operativos y técnicos en información estratégica para la toma de decisiones y el monitoreo de políticas públicas ambientales.

## 📊 Fuentes de Datos
Para asegurar la integridad y profundidad del análisis, se han consolidado las siguientes fuentes públicas:
* **Agencia de Protección Ambiental (APRA):** Datos históricos desde 2009 hasta la actualidad, centrados en indicadores de emisiones y concentración de contaminantes en estaciones de monitoreo urbano.
* **Autoridad de Cuenca Matanza Riachuelo (ACUMAR):** Datos provenientes del Plan Integral de Saneamiento Ambiental (PISA), aportando una visión crítica sobre el impacto y las intervenciones en la Cuenca del Río Matanza-Riachuelo.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python
* **Librerías de Procesamiento:** Pandas, NumPy
* **Librerías de Visualización:** Matplotlib, Seaborn
* **Entorno de Desarrollo:** Google Colab / Jupyter Notebooks
* **Conceptos Clave Aplicados:** Limpieza de datos (Data Wrangling), Análisis Exploratorio de Datos (EDA), Control de Calidad de Datos (Data QA), Análisis de series temporales.

## 🎯 Objetivos del Análisis
1. **Evaluación de Tendencias:** Identificar la evolución temporal de los principales indicadores de calidad del aire en la última década.
2. **Correlación Inter-Institucional:** Cruzar los datos de APRA con las métricas de saneamiento de ACUMAR para evaluar el impacto de las intervenciones ambientales conjuntas.
3. **Identificación de Desviaciones:** Detectar anomalías, valores atípicos (outliers) o picos de contaminación vinculados a eventos estacionales o zonas geográficas críticas.
